NDR Core API

The NDR Core API implementation allows you to connect to another NDR Core instance, enabling federated search across multiple NDR Core installations. This is useful for distributed data repositories or collaborative research projects.

Status: Forthcoming - This API implementation is under active development. Full documentation will be available in a future release.
Use Cases

  • Search across multiple institutional repositories
  • Aggregate results from distributed NDR Core installations
  • Create meta-search interfaces for collaborative projects
