Search & API Configuration

NDR Core connects to external data sources through APIs and provides a search interface for that data. This page shows you how to configure and customize search functionality.

Quick Workflow

  1. Configure API connection
    → Search Configuration > Create Search Configuration
  2. Define search fields (optional - simple search is automatic)
    → Search Configuration > Manage Search Fields
  3. Create custom search form (optional)
    → Search Configuration > Edit Search Form
  4. Configure result display
    → Result Configuration
  5. Create search page
    → Pages > Create Search Page

Common Tasks

Create a simple search (single search box):

  1. Create a Search Configuration with API connection
  2. No search fields needed - simple search is automatic
  3. Create a search page and select the configuration

Create an advanced search (multiple fields):

  1. Create a Search Configuration with API connection
  2. Create individual search fields
  3. Compose fields into a search form
  4. Create a search page and select the configuration

Tip: Start with a simple search and add fields as needed. Test your configuration on the Test Page before creating the final search page.
